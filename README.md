## Como rodar localmente com sincronização remota

1. Instale dependências:

```bash
npm install
```

2. Inicie o servidor:

```bash
npm start
```

3. Acesse o site no navegador:

```bash
http://localhost:3000
```

4. Para acessar de outros celulares na mesma rede, use o IP do computador que está rodando o servidor, por exemplo:

```bash
http://192.168.x.x:3000
```

> As alterações feitas pelos admins agora são salvas no servidor e passam a ser compartilhadas entre dispositivos, com fallback local quando não houver conexão ao servidor.
